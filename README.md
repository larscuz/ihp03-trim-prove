# IHP03-01 prøve – Trim AS (bokmål)

Kjør lokalt:

```bash
npm install
npm run dev
```

Åpne http://localhost:3000

- Kandidat skriver navn øverst
- 3 faner: Fagprøve / Kompetansebevis / Informasjon
- PDF-knapp lager PDF av aktiv fane

NB: UDIR-læreplanen for IHP03-01 er publisert på nynorsk. Teksten i denne løsningen er oversatt til bokmål for bruk i prøve.
